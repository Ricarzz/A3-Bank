# A3-Bank

Este repositório contém os principais pontos de segurança do sistema bancário A3-Bank, desenvolvido como parte do projeto de A3 de Programação e Soluções Computacionais.

## Funcionalidades incluídas:

 **Login**
- Bloqueios após tentativas incorretas
- Recuperação de senha com verificação por e-mail
- Armazenamento seguro de senhas (Hash SHA-256)

 **Transferências**
- Notificações
- Validações completas de dados
- Confirmação e avisos para valores altos (acima de R$1.000)
- Bloqueio de operações suspeitas em horários não permitidos
- Detecção de múltiplas transferências em curto prazo

 **Denúncias**
- Alerta ao tentar transferir para contas com denúncias
- Bloqueio automático após 5 denúncias

## Tecnologias
- Java (backend)
- MySQL (banco de dados)
- Swing (interface gráfica)
- NetBeans (IDE)
